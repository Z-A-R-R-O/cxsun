# Tirupur Connect

Tirupur Connect is registered as a separate tenant feature under `apps/frontend/src/features/tirupur-connect/`.

- Company feature flag: `feature-tirupur-connect`
- Tenant pages: `app-tirupur-connect-*`
- Public page: `/tirupur-connect`
- Backend API: `/api/v1/tirupur-connect`

The current implementation is the foundation surface: overview, settings, route wiring, feature hiding, and the public landing route. Full supplier, buyer, product, RFQ, messaging, membership, event, news, and admin CRUD remain tracked in `assist/execution/task.md`.
